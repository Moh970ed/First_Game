# First Game - Godot Platformer

A professional 2D platformer game built using the **Godot Engine**. This project demonstrates core game mechanics, asset management, and structured scene organization.

## 🚀 Features
- **Smooth Character Movement**: Responsive controls for a great player experience.
- **Scene Management**: Organized structure for levels, players, and UI.
- **Asset Integration**: High-quality sprites and sounds.
- **Extensible Code**: Clean GDScript following best practices.

## 🛠️ Built With
- [Godot Engine](https://godotengine.org/) - The game engine used.
- [GDScript](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_basics.html) - Main scripting language.

## 📂 Project Structure
- `assets/`: Contains all visual and audio resources.
- `scenes/`: Godot scene files (.tscn).
- `scripts/`: GDScript logic files (.gd).
- `project.godot`: Main project configuration.

## 🎮 How to Run
1. Download and install [Godot Engine](https://godotengine.org/download).
2. Clone this repository:
   ```bash
   git clone https://github.com/Hozyfa-Batisha/First-Game.git
   ```
3. Open Godot and click **Import**.
4. Navigate to the cloned folder and select `project.godot`.
5. Click **Import & Edit** and press **F5** to play!
