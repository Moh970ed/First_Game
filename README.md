## 🎯 Game Mechanics

### 🎮 Controls

- **← →** : Move Left/Right

- **Space** : Jump

- **Movement Speed**: 150 units/sec

- **Jump Height**: -300 velocity

### 🛠️ Gameplay Elements

- **Coins**: Touch to collect and remove from the scene.

- **Platforms**:
  - **Green**: Move horizontally to add challenge.
  - **Blue**: Allow one-way jumping through them.

- **Enemies**: Patrolling enemies that the player must avoid.

- **Killzone**: Touch it and respawn after a short delay!

## 🚀 Getting Started

To run and develop this project, follow these steps:

### Prerequisites

Ensure you have **Godot Engine 4.x** or later installed.

### Installation

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/Mohamed-Elsayed970/Game.git
   ```

1. **Navigate to the Directory:**

   ```bash
   cd Game
   ```

1. **Open the Project:**
  - Open the Godot Engine.
  - Click **Import** and select the `project.godot` file inside the directory.
  - Click **Import & Edit**.
